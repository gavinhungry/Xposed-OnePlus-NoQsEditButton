Xposed-OnePlus-NoQsEditButton
=============================
Xposed module to hide the Quick Settings edit button on OxygenOS.

Tested on OnePlus 2, but may work with other OxygenOS devices.

Build
-----

    $ ant build

Install
-------

    $ adb install dist/NoQsEditButton.apk
